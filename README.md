## ✨ Features

### 📊 Dashboard
- Daily greeting with real-time date
- Live stats: Calories, Workouts, Water, Streak
- Today's workout plan with completion tracking
- Animated progress ring for daily goals

### 🏋️ Workout Library
- **28 real exercises** across 8 categories:
  - Chest, Back, Legs, Arms, Shoulders, Core, Cardio
- Detailed exercise modal with sets, reps, calories, and step-by-step instructions
- Filter by muscle group
- Log workouts with one click

### 🧰 Fitness Tools
- **BMI Calculator** — Calculate Body Mass Index with health category
- **Calorie Counter** — Track daily food intake with add/remove
- **Workout Timer** — Precision stopwatch with lap tracking
- **Water Tracker** — Visual glass tracker (8 glasses/day goal)
- **One Rep Max Calculator** — Epley formula estimation
- **Body Fat Estimator** — US Navy method for males & females

### 📈 Progress Tracking
- Weekly bar charts (Calories Burned & Workout Minutes)
- Weight log with trend tracking
- **12 unlockable achievements** with milestone badges

### 👤 Profile & Settings
- Personal stats overview (total workouts, calories, streak, minutes)
- Customizable goals (calorie goal, workout goal, target weight)
- Toggle settings (notifications, dark mode, reports, sounds)

### 📱 Mobile Responsive
- Bottom navigation bar for mobile devices
- Touch-friendly interface
- Fully responsive layout

### 📢 Ad Placements
- Banner ads with CTAs
- Inline sponsored content

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Victorezhil310/VictorFit.git
   ```

2. **Open in browser**
   ```bash
   cd VictorFit
   open index.html
   ```

No build tools or dependencies required — pure HTML, CSS & JavaScript!
